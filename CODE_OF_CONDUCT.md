# Code of Conduct

This project adopts the **Contributor Covenant, version 2.1**, as its
Code of Conduct.

The full text of the Contributor Covenant 2.1 is available at:

  <https://www.contributor-covenant.org/version/2/1/code_of_conduct/>

By participating in this project — as a contributor, issue reporter,
reviewer or any other role — you agree to abide by its terms.

## Scope

This Code of Conduct applies to all project spaces, including:

- The GitHub repository (issues, pull requests, discussions, code
  reviews, commits).
- Any communication channel formally associated with the project.
- Public spaces when an individual is representing the project (e.g.,
  using an official project email address, posting from an official
  account, or acting as an appointed representative at an event).

## Reporting

If you observe behaviour that may violate this Code of Conduct, please
report it by contacting the project maintainer through the contact
listed in the project's `LICENSE` file. All complaints will be reviewed
and investigated promptly and fairly.

The maintainer is obligated to respect the privacy and security of the
reporter of any incident.

## Enforcement

Maintainers are responsible for clarifying and enforcing the standards
of the Contributor Covenant and will take appropriate and fair action
in response to any behaviour deemed inappropriate, threatening,
offensive or harmful.

Consequences may include — depending on severity — a private warning, a
public clarification, a temporary or permanent ban from the project
spaces, or any other action the maintainer deems appropriate.
